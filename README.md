# Database Packager

Simple software developed to make it easy to extract database updates for SITA - Insight system.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

At least Java 8 is required to run this program. Besides that, you'll need valid connection to svn desired.

You can find latest Java version at: https://www.java.com/en/download/

### Installing

Install Java 8 and use it in Eclipse. Other jars are embedded.
